# **App Name**: EduZenith

## Core Features:

- Student Authentication: Secure login for students using email and password with Firebase Authentication.
- Teacher Authentication: Secure login for teachers using email and password with Firebase Authentication.
- Assignment Creation: Teachers can create new assignments with descriptions, due dates, and file uploads.
- Assignment Listing & Submission (Student): Students can view a list of assignments, due dates, and upload completed assignments. Assignment status will also be shown.
- Assignment Tracking (Teacher): Teachers can view a list of all students and track their assignment submissions.
- Grading & Feedback: Teachers can grade submitted assignments and provide comments or feedback.
- AI-Powered Assignment Recommendation: Teachers can use a tool that generates assignment suggestions based on course topics. This will also automatically generate instructions based on files.

## Style Guidelines:

- Primary color: HSL values of H=220, S=60%, L=50% (Hex: #3D85E6) to create a modern and trustworthy feel. 
- Background color: HSL values of H=220, S=20%, L=90% (Hex: #E2EBF5) for a light and clean backdrop.
- Accent color: HSL values of H=190, S=50%, L=40% (Hex: #33A3A3) to draw attention to important interactive elements and notifications.
- Body and headline font: 'Inter' for a clean, modern, and readable interface.
- Use simple, outline-style icons to represent different actions and sections.
- Use a clean, grid-based layout to organize content. Focus on readability with ample whitespace.
- Subtle transitions and animations to provide feedback and guide the user through the website.
